## bulid文件
  - index.js webpack配置文件[很少修改]
## mock
  - mock数据的文件夹[模拟数据]，实际开发使用的是真接口，
## public
  - ico图标，静态页面，即静态资源，webpack编译不会编译这个文件，
## src 
  - 在此编写代码的地方
  - api 文件 请求现骨干的文件
  - assets文件夹 放置共享的静态资源，webpack打包时，会进行编译
  - components 文件夹，放置非路由组件的
  - icons文件 放置svg矢量图
  - layout 放置了组件和混入mixin
  - router
